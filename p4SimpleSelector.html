Let's Begin

Simple Selector can achived by using tag, id, and class In tag, we give style to tag in style in head it will applies to the all tags present in body.

In id we use id attribute in a specific tag that we want to apply Style. We use # to specify id in style. example this paragaraph id is p2. So we give (#id name)#p{ css code }.this paragraph is chaned using id.

In class we use class attribute in a specific tag that we want to apply Style. We use . to specify id in style. tags having same class name will effect the style example this paragaraph class is c3 So we give (.classname) .c3{ css code }. this paragraph is chaned using class.
